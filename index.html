Use Postman collections or curl from our examples.

Welcome to Open GI Shell

Browser‑based try‑outs coming soon. For now, use curl/Postman with your sandbox keys.

# 1) Obtain token
curl -X POST https://sandbox.api.yourapi.com/oauth/token \
  -d "client_id=YOUR_CLIENT_ID" \
  -d "client_secret=YOUR_SECRET"

# 2) Get a quote (minimal)
curl -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"risk":{"driverAge":30,"carValue":8000}}' \
  https://sandbox.api.yourapi.com/quotes

# 3) Accept quote → policy
curl -X POST -H "Authorization: Bearer <token>" \
  https://sandbox.api.yourapi.com/quotes/q_123/accept
